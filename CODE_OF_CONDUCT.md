# Contributor Covenant Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in Pizza Logs
a harassment-free experience for everyone, regardless of age, body size,
visible or invisible disability, ethnicity, sex characteristics, gender identity,
level of experience, education, socioeconomic status, nationality,
personal appearance, race, religion, or sexual identity and orientation.

We pledge to act and interact in ways that contribute to an open, welcoming,
diverse, inclusive, and healthy community.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Be respectful and constructive
- Assume good intent
- Accept feedback gracefully
- Focus on improving the project
- Help others learn

Examples of unacceptable behavior:

- Harassment or discrimination
- Trolling or insulting comments
- Personal or political attacks
- Public or private harassment
- Sharing others' private information
- Malicious code, spam, or intentionally harmful submissions

## Enforcement Responsibilities

Project maintainers may remove, edit, or reject comments, commits, code,
issues, or other contributions not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies in:
- GitHub issues
- Pull requests
- Discussions
- Community chat spaces
- Any official project spaces

## Enforcement

Report concerns by opening a private security report or contacting maintainers.

## Enforcement Guidelines

Possible actions include:

1. Correction
2. Warning
3. Temporary ban
4. Permanent ban
