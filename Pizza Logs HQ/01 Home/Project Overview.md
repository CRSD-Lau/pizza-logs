# Project Overview

## Product Name
Pizza Logs

## What It Is
A modern World of Warcraft combat log analytics platform.

## Built For
PizzaWarriors first.

## Future Potential
Any WoW guild / community.

## Core Features
- Upload combat logs
- Parse raid bosses only
- Weekly rankings
- Boss leaderboards
- Player history
- Milestones
- Historical stats
- Deduplication

## Current Stack
- Next.js
- React
- TypeScript
- Tailwind
- PostgreSQL
- Prisma
- Python parser
- Railway
